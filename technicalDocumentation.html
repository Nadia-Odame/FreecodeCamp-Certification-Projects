Introduction

Python is a high-level, interpreted programming language known for its clear syntax and readability. Created by Guido van Rossum and first released in 1991, Python has grown into one of the most popular languages in the world.

Python supports multiple programming paradigms including procedural, object-oriented, and functional programming. Its design philosophy emphasizes code readability with the use of significant indentation.

Python is widely used in web development, data science, artificial intelligence, scientific computing, and automation. Its vast standard library and active community make it a go-to choice for beginners and experts alike.

# Your first Python program print("Hello, World!")

Python files use the .py extension. You can run a Python script from the terminal using the python command followed by the filename.

Variables and Types

In Python, variables are created when you assign a value to them. Python is dynamically typed, meaning you do not need to declare the type of a variable explicitly.

Python has several built-in data types. The most commonly used are integers, floats, strings, and booleans. Python automatically infers the type based on the assigned value.

name = "Alice" # str age = 30 # int height = 5.7 # float is_active = True # bool

You can check the type of any variable using the built-in type() function. Type conversion between compatible types is also straightforward in Python.

  • int — whole numbers like 1, 42, -7
  • float — decimal numbers like 3.14, -0.5
  • str — text enclosed in quotes
  • bool — either True or False
Control Flow

Control flow statements allow you to control the order in which code is executed. Python uses if, elif, and else statements for conditional logic.

Indentation is critical in Python — it defines the scope of code blocks. Unlike many other languages, Python does not use curly braces to group statements.

score = 85 if score >= 90: print("Grade: A") elif score >= 75: print("Grade: B") else: print("Grade: C")

Python also provides for and while loops for iteration. The for loop is especially powerful when combined with Python's iterable types like lists, strings, and ranges.

for i in range(5): print(f"Count: {i}")
  • Use break to exit a loop early
  • Use continue to skip the current iteration
  • Use pass as a placeholder in empty blocks
Functions

Functions are reusable blocks of code defined using the def keyword. They help organize code, avoid repetition, and improve readability.

Python functions can accept parameters and return values. Default parameter values and keyword arguments make Python functions very flexible.

def greet(name, greeting="Hello"): return f"{greeting}, {name}!" print(greet("Bob")) print(greet("Alice", "Hi"))

Python also supports lambda functions — small anonymous functions defined with the lambda keyword. They are useful for short, throwaway operations.

Functions in Python are first-class objects, meaning they can be passed as arguments, returned from other functions, and stored in variables.

  • Use *args to accept variable positional arguments
  • Use **kwargs to accept variable keyword arguments
Lists and Dictionaries

Lists are ordered, mutable collections that can hold items of any type. They are one of the most used data structures in Python and support a wide range of built-in methods.

fruits = ["apple", "banana", "cherry"] fruits.append("mango") print(fruits[0]) # apple

Dictionaries store data as key-value pairs. They are unordered (in older Python versions) and optimized for fast lookups by key. Since Python 3.7, dictionaries maintain insertion order.

person = { "name": "Alice", "age": 30, "city": "Paris" } print(person["name"]) # Alice

Both lists and dictionaries support comprehensions — a concise syntax for creating new collections based on existing ones.

  • Access list items using zero-based index
  • Use .append() to add to a list
  • Use .get() on dicts to avoid KeyError
Modules and Packages

A module is a file containing Python code — functions, classes, or variables — that can be imported and reused in other scripts. Python comes with a rich standard library of built-in modules.

Packages are directories of modules with an __init__.py file. Third-party packages can be installed using pip, Python's package manager, from the Python Package Index (PyPI).

import math print(math.sqrt(16)) # 4.0 from datetime import datetime print(datetime.now())

You can also create your own modules simply by saving Python code in a .py file and importing it into another script using the import statement.

  • Use import module to import an entire module
  • Use from module import name for specific items
  • Use as to create an alias on import
